# Product Overview

Gesture Audio Canvas は、キャンバス上のジェスチャー操作で音声を直感的に編集・再生できるWebアプリケーションです。ユーザーは描画した線の形状によって音声の再生時間、方向（順再生/逆再生）、ピッチを制御できます。

## Core Capabilities

- **ジェスチャーベース音声操作**: マウス/タッチでキャンバスに線を描き、その形状で音声パラメータを決定
- **リアルタイム音声加工**: Web Audio APIによる再生速度・ピッチ・方向のリアルタイム変換
- **Firebase認証連携**: メール/パスワード認証によるアクセス制御
- **視覚フィードバック**: 描画軌跡のリアルタイム表示

## Target Use Cases

- 音声の直感的な探索・実験
- ジェスチャーによるインタラクティブな音声表現
- 教育・デモンストレーション用途

## Value Proposition

従来のスライダーやボタン操作ではなく、身体的なジェスチャーで音声を操作することで、より直感的で表現力のある音声体験を提供します。
